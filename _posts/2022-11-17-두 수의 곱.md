## *< 두 수의 곱 >*
<br>

> #### *문제 설명*
#### *정수 num1, num2가 매개변수 주어집니다. num1과 num2를 곱한 값을 return 하도록 solution 함수를 완성해주세요.*
<br>

> #### *제한사항*
#### *0 ≤ num1 ≤ 100,  - 0 ≤ num2 ≤ 100*
<br>

> #### *입출력 예*

|num1|num2|result|
|:---:|:---:|:---:|
|3|4|12|
|27|19|513|
|50|30|1500|

<br>

> #### *해설*
#### num1과 num2를 입력받아 두 수의 곱을 구하는 문제입니다.
<br>

> #### *풀이*
```python
def solution(num1, num2):
    answer = 0
    answer = num1 * num2
    return answer
```
